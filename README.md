# Java DSA Practice

This repository contains my personal practice of Data Structures and Algorithms using Java. I’ve been solving problems regularly to improve my problem-solving skills and get better at writing efficient code.

Most of the problems here are from platforms like LeetCode, GeeksforGeeks, and Striver’s DSA Sheet. Instead of just solving them once, I try to understand the logic properly and then write clean and simple code.

# What you’ll find here

* Solutions to common DSA problems
* Different approaches (brute force + optimized where possible)
* Topics like arrays, strings, recursion, searching, sorting, etc.
* Code written in a simple and readable way

# How I approach problems

Whenever I solve a problem, I usually follow this process:

* First, I understand the problem statement clearly
* Then I think of a basic (brute force) solution
* After that, I try to optimize it
* Finally, I write clean code and test it

This approach has really helped me improve my thinking and consistency.
